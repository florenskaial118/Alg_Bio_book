# algbio_book

Конспекты занятий магистратуры МФТИ ФБМФ «Алгоритмическая биология»

## О проекте

Этот репозиторий содержит конспекты лекций и семинарских занятий магистерской программы «Алгоритмическая биология» МФТИ. Материалы охватывают два семестра и включают теоретические основы, практические задания и примеры кода.

## Содержание

### 1 семестр

- Базы данных в биоинформатике
- NGS и филогенетика
- Практика по клинической интерпретации геномных данных
- Практика по алгоритмам и структурам данных

### 2 семестр

- Алгоритмы в биоинформатике
- Машинное обучение
- Медицинская генетика
- Статистика
- DevOps

## Локальный запуск

### Требования
- Quarto (установка: quarto.org)

### Сборка книги

```bash
# Клонирование репозитория
git clone https://github.com/florenskaial118/Alg_Bio_1_sem.git
cd Alg_Bio_1_sem

# Сборка HTML-версии
quarto render

# Просмотр в браузере (локальный сервер)
quarto preview
```
